# 自定义线路提醒Plus 插件 line_status_tip_plus

## 插件简介

当用户流量耗尽或套餐到期时，订阅接口会下发多条自定义线路，用于在客户端醒目提醒用户“流量已用完或套餐已到期，请及时续费”。

管理员可在后台自由填写多行提示内容，灵活配置不同提醒文本。  
适用于所有基于 XBoard的订阅系统，提升用户体验，减少因订阅无响应导致的疑惑。

---

## 功能特性

-   用户流量耗尽或套餐到期时，自动下发多条自定义线路  
-   支持多行自定义提醒内容，每行生成一条线路  
-   兼容所有主流客户端（如 Clash、Shadowrocket、Surge 等）  
-   支持插件化扩展，可与其他订阅策略灵活配合  

---

## 配置项说明（config.json）

| 配置项      | 类型 | 默认值 | 说明 |
| ------------ | ---- | ------ | ---- |
| tip_lines    | text | （空） | 提醒线路内容，每行代表一条自定义线路名称 |
