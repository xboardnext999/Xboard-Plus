# 📒 Xboard NEXT 插件

    自定义后台接口，对接Xboard NEXT管理后台和客户端。

---

## ✨ 功能说明
- **登记内容**：
- 
  ##### 1、用户接口
  ###### 1.1、查询购买套餐对应的节点
  ###### 1.2、查询邀请页面的提现费率
  ###### 1.3、添加充值订单接口(与套餐使用一套支付方式)
- 
  ##### 2、管理员接口
  ###### 2.1、后台管理查询套餐分组（配置自定义套餐）
  ###### 2.2、后台管理查询用户（发送消息、配内置代理）
  ###### 2.3、后台管理查询优惠码（发送消息）
  ###### 2.4、后台管理仪表盘（仪表盘）
  ###### 2.5、后台管理仪表盘节点、用户流量排行（仪表盘）
  ###### 2.6、后台管理节点轮询（更新备用节点）

---

## 📦 安装与使用
- **安装插件**：将压缩包上传到插件页面。
- **启用插件**：在后台插件管理界面启用插件。
